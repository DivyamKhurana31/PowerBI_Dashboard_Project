# Royal Enfield GT 650 Sales Performance Dashboard (Power BI)
This project is an interactive Power BI dashboard developed to analyze the sales performance of the Royal Enfield GT 650 model. The dashboard provides key insights into revenue, sales trends, profit, and product performance using visually engaging charts and KPI indicators.

## Dashboard Overview

The dashboard provides insights into:
- Total Revenue generated
- Units Sold
- Total Profit
- Total Expenses
- Monthly Sales Trends
- Revenue by State
- Revenue by Variant
- Product information such as Engine Capacity, Mileage, and Top Variant Price

## Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Visualization
- Data Cleaning
- Data Analysis
